.. _installation:

============
Installation
============

There are several ways to require and install this extension. We recommend to get this extension via
`composer <https://getcomposer.org/>`__.

.. _installation-composer:

Via Composer
============

If your TYPO3 instance is running in composer mode, you can simply require the extension by running:

.. code-block:: bash

   composer req leuchtfeuer/typo3-audience-studio

.. _installation-extensionManager:

Via Extension Manager
=====================

Open the extension manager module of your TYPO3 instance and select "Get Extensions" in the select menu above the upload
button. There you can search for `audience studio` and simply install the extension. Please make sure you are using the latest
version of the extension by updating the extension list before installing the Audience Studio extension.

.. _installation-zipFile:

Via ZIP File
============

You need to download the Audience Studio extension from the
`TYPO3 Extension Repository <https://extensions.typo3.org/extension/audience_studio/>`__ and upload the zip file to the extension
manager of your TYPO3 instance and activate the extension afterwards.

.. important::

   Do not forget to activate the extension in the TYPO3 Extension Manager.
