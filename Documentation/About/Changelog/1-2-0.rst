==========================
Version 1.2.0 - 2021/01/20
==========================

This is the initial TER release of this extension which is now compatible for "classic" TYPO3 installations. Besides of that, it
is now possible to transfer the Audience Studio user ID from browser's local storage to a dedicated cookie.

Download
========

Download this version from the `TYPO3 extension repository <https://extensions.typo3.org/extension/audience_studio/>`__ or from
`GitHub <https://github.com/Leuchtfeuer/typo3-audience-studio/releases/tag/v1.2.0>`__.

Added
=====

* Documenation
* Support for non-composer setups
* Transer user ID from :ref:`local storage <installation-controlTagAndCookies-cookieValueFromLocalStorage>` to cookie
